Reproductor de Música en Python

Reproductor de Música es una aplicación de escritorio desarrollada en Python que permite reproducir archivos MP3 de forma sencilla mediante una interfaz gráfica amigable.

El proyecto utiliza PySimpleGUI para la interfaz gráfica y pygame para la reproducción de audio.

Características

⦁	Reproducción de archivos MP3
⦁	Selección de carpetas con múltiples canciones
⦁	Selección de una canción individual
⦁	Controles de reproducción:

1.	Reproducir / Pausar
2.	Siguiente canción
3.	Canción anterior

⦁	Control de volumen en tiempo real
⦁	Visualización de un logo central
⦁	Uso de Threads para evitar que la interfaz se congele

Tecnologías utilizadas

⦁	Python 3.8+
⦁	pygame (reproducción de audio)
⦁	PySimpleGUI (interfaz gráfica)
⦁	threading (ejecución en segundo plano)
⦁	os (manejo de archivos y rutas)
